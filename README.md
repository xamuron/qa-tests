# QA Tests

Статический тренажер для подготовки QA-инженеров. Сейчас доступен тест PSB из 35 вопросов.

## Локальный запуск

Откройте `index.html` в браузере или запустите любой статический HTTP-сервер, например:

```bash
python3 -m http.server 8000
```

После этого приложение будет доступно по адресу `http://localhost:8000`.
